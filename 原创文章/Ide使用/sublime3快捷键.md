# 文件操作，格式，粘贴
## 复原文件到保存状态
    File选项卡 + Revet File
    快捷键：Alt + F => V
## 带缩进粘贴文本
    Ctrl + Shift + V
## 从历史粘贴
    Ctrl + K => Ctrl + V
# 行操作
## 多行同时操作
    Ctrl + Shift + L 等同于 Ctrl + Shift + 右键往下拖
## 块状注释
    Ctrl + Shift + /
## 在光标行下插入一空行
    Ctrl + Enter
## 在光标行上插入一空行
    Ctrl + Shift + Enter
# 行内删除
## 向后删除单词（单词的界定应该是空格，点等字符）
    Ctrl + Delete
## 向前删除单词
    Ctrl + Backspace
## 从光标点删除字符到行尾
    Ctrl + K => Ctrl + K
## 从光标点删除字符到行首
    Ctrl + K => Ctrl + Bacspace
# html标签
## 使用标签包裹选中内容（默认使用P标签)
    Alt + Shift + W    
# 打标记用于快速操作
1. 设置标记Edit选项卡 => Mark
2. 从光标位置到标记选中
3. 从光标位置到标记删除   

# 代码折叠
## 折叠选中行所属的代码块
    Ctrl + Shift + [ 
## 展开选中行所属的代码块
    Ctrl + Shift + ] 
## 折叠所有html属性
    Ctrl + k => Ctrl + t
## 展开所有折叠
    Ctrl + k => Ctrl + j
# 选择内容
## 选择词
    Ctrl + D
## 选择行
    Ctrl + L
## 选择标签内容
    Ctrl + Shift + A
## 选择范围(单词，圆括号，中括号，花括号)
    Ctrl + Shift + space
## 选择括号内容
    Ctrl + Shift + M
## 按缩进选择
    Ctrl + Shift + J
## 设置操作点
    Ctrl + k => ctrl + S
## 从光标位置选择到操作点
    Ctrl + K => ctrl + A
# 查找
## 查找下一个
    F3
## 查找上一个
    Shift + F3
## 快速查找(光标自动匹配词，移动到下一个匹配元素并包裹）
    Ctrl + F3
    不用按Ctrl + F，光标移动查找元素并选中，注意别不小心替换了
## 快速查找所有
    Alt + F3
## 快速添加
    Ctrl + D
## 文件中查找
    Ctrl + Shift + F
# 跳转
## 跳转到定义的函数或类
    Ctrl + R
## 跳转到定义
    F12
## 跳到上一个光标位置
    Alt + -
## 跳到下一个光标位置（跳到上一个位置再跳回来)
    Alt + Shift + -
## 滚轮滑动到选中的文字
    Ctrl + K => Ctrl + C
## 增加书签
    Ctrl + F2
## 上一个书签
    Shift + F2
## 清除所有书签
    Ctrl + Shift + F2
## 跳到匹配的的括号位置
    Ctrl + M
# 不常用
## 复制上一个操作
    Ctrl + Y
## 向右对齐缩进
    Ctrl + ]
## 向左删除缩进
    Ctrl + [
## 把下面的行连接到这一行 
    Ctrl + J
## 行排序
    F9
## 大小写敏感行排序
    Ctrl + F9
## 转大写
    Ctrl + k => Ctrl + U
## 转小写
    Ctrl + k => Ctrl + L

## 隐藏左侧的菜单
    ctrl + k + b
